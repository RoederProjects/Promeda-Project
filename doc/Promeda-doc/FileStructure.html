File Structure

 

     Promeda.local-working-dir

     Promeda-doc    // documentation files

     Promeda-bin

     dist       // current local media archive location (see also app.properties)

     java      // installation files for java jre 6 (Windows, Mac) and jre 8 (Windows)

     Jdk6.exe

     Jdk6

     node_modules // node.js + npm (ref: https://docs.npmjs.com/files/folders)

     properties

     app.properties  // basic app properties

     banner

     banner.properties // banner templates (name, dimension)

     stores

     storename.properties   // ftp-connection, product image sizes

     Promeda.jar     // executable JAR (double-click to start the application)
